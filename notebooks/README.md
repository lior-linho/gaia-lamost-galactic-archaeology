# Notebooks

This folder contains exploratory and analysis notebooks for the Gaia–LAMOST Galactic Archaeology Project.

## Purpose

The notebooks document the main scientific workflow, including data querying, catalogue exploration, cross-matching, feature construction, visualization, and later machine learning experiments.

## Notebook Organization

The notebooks are organized by project milestone:

    01_gaia_dr3_query.ipynb
    02_lamost_catalogue_exploration.ipynb
    03_chemo_kinematic_features.ipynb

Additional notebooks may be added as the project develops.

## Notes

Notebooks are used for exploration, validation, and figure generation.

Reusable functions and stable analysis code should gradually be moved into the `src/` folder to improve reproducibility and maintainability.

## Recommended Practice

Each notebook should include:

- a short goal at the beginning
- clear section headings
- comments explaining important steps
- saved outputs when relevant
- links to related milestone notes or figures
