# Project III — Stellar Population Analysis

## Scientific Question

Which Galactic stellar populations do the Gaia-LAMOST candidate stars most likely belong to?

## Background and Motivation

After candidate stars are identified and their orbital properties are estimated, the next scientific step is to interpret them as possible members of Galactic components or merger-related structures. This project connects orbital dynamics, metallicity, and velocity-space evidence to stellar-population classification.

## Data and Methods

Target populations include:

- Thin disk
- Thick disk
- Stellar halo
- Gaia-Sausage-Enceladus
- Helmi Stream
- Sequoia
- Nyx
- Splash
- Other dynamically unusual populations

Core methods will include:

- Disk-halo separation
- Prograde and retrograde classification
- High-velocity candidate classification
- Metal-poor candidate classification
- Comparison with literature-defined dynamical regions
- Candidate-level population tagging

This project has not yet started as a standalone analysis. It will begin after the current Project II orbital-family interpretation step is completed.

## Validation and Uncertainty

Population classifications will require careful treatment of:

- Uncertain distances
- Velocity uncertainties
- Orbital-parameter uncertainties
- Overlap between disk, halo, and merger-remnant regions
- Literature-definition differences

## Discussion

The goal of this project is to move from candidate detection to astrophysical interpretation. It asks not only whether stars are unusual, but what Galactic component or accretion event they may trace.
